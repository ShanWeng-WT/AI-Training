🎯 課程目標

  • 看透本質： Agent 與 Chatbot 的差異
  • ReAct 框架：「感知 - 思考 - 行動」迴圈
  • 階段總結： 前 6 堂課技術堆疊

📖 什麼是 Agent？

Chatbot 只能說話，Agent 能夠行動。
  • Chatbot： 預測文字
  • Agent： 擁有目標 + 使用工具 + 自我修正

💡 頓悟時刻

Agent 不是魔法，而是一個 while(true) 迴圈！
  • 觀察 (Observation)
  • 思考 (Thought)
  • 行動 (Action)

🛠️ 核心機制：ReAct 框架

  • Reasoning (推理)： 分析狀況，決定下一步
  • Acting (行動)： 呼叫外部工具

⚠️ 為什麼 Agent 會卡死？

  • 工具不穩定： 陷入無限重試
  • 目標模糊： 不知道何時算「完成」
